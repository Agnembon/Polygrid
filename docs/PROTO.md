# Polytable - Proto

Quick guide to set up your development environment with Proto and Bun.

## 1. Install Proto

### Windows (PowerShell)

```powershell
irm https://moonrepo.dev/install/proto.ps1 | iex
```

### Verify installation

```bash
proto --version
```

## 2. Install Bun using Proto

Once Proto is installed, use it to install Bun:

```bash
proto install bun
```
## 3. Verify installation

```bash
bun --version
```
